# Moonbox与Davinci集成

## Davinci

Davinci项目地址：[Davinci](https://github.com/edp963/davinci)

## moonbox-jdbc jar获取

下载moonbox-jdbc jar，下载地址：[releases]()

## 配置 
   
1. 将moonbox-jdbc jar拷贝到Davinci安装目录的lib路径下
2. 重启Davinci
3. 登录Davinci, 并在Source页面配置Moonbox的JDBC连接信息, 如图
